Fetching Data From PostgreSQL Using Python psycopg2

Published: Sun 04 March 2018
By Zemian Deng

In Blog.

tags: python postgres 

import psycopg2
conn = psycopg2.connect('dbname=test user=test')
cur = conn.cursor()
cur.execute("select * from users order by username")
while True:
    row = cur.fetchmany()
    if len(row) == 0:
        break
    print(row)
cur.close()
conn.close()
