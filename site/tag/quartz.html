New Quartz 2.3.1 Released

Published: Tue 02 April 2019
By Zemian Deng

In Blog.

tags: quartz

Quartz project has little development activities for quite some time now. I finally able to put in some time and help get many bugs fixed up. After more than two years since Quartz last release (April 2017), I am happy to announce the quartz-2.3.1 is finally out!

This release is a minor release that fixed many bugs on the 2.3 series that uses JDK 7. We also cleaned up the documentation that generated from asciidoc and placed it along with the source repository.

Our master branch now target the next quartz 2.4.0 milestone, which will support the JDK 8 build.
