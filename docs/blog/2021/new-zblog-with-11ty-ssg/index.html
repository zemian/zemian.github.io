New ZBlog with 11ty SSG

I changed to another Static Site Generator (SSG) called Eleventy (or 11ty) to create my blog. I kept my previous BulmaCSS theme I made though, so the looks is still about the same.

Few things I like about 11ty:

  • It has the minimal (in fact zero config) to get started! Give a Markdown file and it will genearate you a output!
  • It allow you to structure your data/project in any way you like. But they do have some default naming conventions if you choose to use it.
  • It supports Data Cascade to allow many ways to provide Data. For example, I can easily add a static JSON file and use it in my template!
  • It supports many template libraries. It default to Liquid, but I prefer Nunchucks template library better, since it resemble Twig more and I like that syntax better.
  • It has simple and overridable layout that you can use, even if you don't choose a fany template.
  • It supports pagination and data collection.
  • It seems to be pretty fast. I have about 300 articles and it generated under

Compare to VuePress:

  • The VuePress default is gear toward for Documentation site, and it is strict on structure and the data source it can use to generate the site.
  • It goes through this complex Hydration that I just don't fee comfortable with it. It's like there things you just don't see or get to understand behind the scene.
  • When you make a simple article change, it regenerate lots of files, including JS file. It feels little weird.
